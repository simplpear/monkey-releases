Description of Service

MonKey is a sandboxed macOS utility providing:

  • Keyboard layout autocorrection — detects and fixes text typed in the wrong keyboard layout (unlimited)
  • Translation — translates selected text between languages using cloud AI (2/day free, unlimited with PRO; offline fallback via Apple Translation)
  • Grammar & style checking — identifies errors in selected text using cloud AI (2/day free, unlimited with PRO; offline fallback via macOS spellcheck)
  • AI Wizard — runs custom AI-powered actions on selected text (PRO only)
